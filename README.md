# To Do List

> In this project I made a to list by usign JS, webpack.

![screenshot](./app_screenshot.png)


## Built With

- HTML&CSS, JavaScript
- Webpack

## Getting Started

For Cloning Repository:

- git clone git@github.com:luftedar/todo-list.git

- cd into todo-list

For Start Webpack

- npm install

- npm start


### Prerequisites

- Webpack, Internet Browser, Note Editor
